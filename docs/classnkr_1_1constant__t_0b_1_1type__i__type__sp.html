Checks if the passed in type is any qualification of any instantiation of nkr::constant_t regardless of its template arguments.

Fulfills the generic need to determine if two potentially different types come from the same template. Because nkr::constant_t$::type_i_type_sp::type_t is passed in to instantiate this interface implementation, this essentially acts as an equality operator for templates with types as operands.

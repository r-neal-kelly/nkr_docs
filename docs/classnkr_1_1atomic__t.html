Detailed Description

template<typename value_p>
class nkr::atomic_t< value_p >

The default template for nkr::atomic_t, which invalidates types not matching any specialization.

This type is essentially a wrapper for the functions contained in nkr::os::atomic. For invalid types, this template provides compile time failure or even pre-compile time failure if using intellisense.

  • nkr::atomic_t<integer_p>

  • nkr::atomic_t<void_t*>

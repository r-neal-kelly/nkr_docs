Detailed Description

template<integer_tr integer_p>
class nkr::atomic_t$::integer_sp< integer_p >

A specialization for integers to be used atomically.

It depends on the operating system, but most integers that are anywhere from the size of the byte_t upto and including the size of the word_t are supported, including both signed and unsigned variants.

Note
There is a difference between the assignment operator of atomic variables and that of normal variables such that with normal variables, trying to assign a real number to an integer will not statically cast the real to the type being assigned, whereas with atomics, this is necessarily going to happen in order to get the full benefits of operating system calls which require both operands to be of the same basic type. Every single operator will statically convert each right-hand-side operator to match the other.
